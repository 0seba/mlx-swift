## Overview

Example software for ``MLX`` and `MLXNN`.

### LinearModelTraining

A simple linear model and a training loop.

### llm-tool

A command line tool for generating text using a Llama / Mistral model.

### mnist-tool

A command line tool for training an MNIST (MLP) model.
